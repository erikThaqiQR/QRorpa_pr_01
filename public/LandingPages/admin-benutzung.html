03. Bestellverwaltung


1. Anzeige der Tischpositionen je nach Organisation des Restaurants. Benachrichtigung durch rote Farbe auf der jeweiligen Tabelle
2. Klicken Sie auf das Menü Bestellen
3. Öffnen der Bestellung auf dem Tisch, der eine neue Bestellung hat
4. Senden der Bestellung im Wartevorgang über die Schaltfläche Wartenschleife

4. Auswahl des Kellners, der die Bestellung übernimmt. Auswahl des Bestellstatus

5. Der Status der Bestellung ändert sich

Sie können Bestellungen auch auf der Seite Tabellenstatus verwalten, indem Sie oben auf die Schaltfläche Tische klicken.
Hier können Sie auf die entsprechende Tabelle klicken, um eine Bestellung zu buchen. Der Status dieser Nachricht ist derzeit nicht definiert.

4. Tabellenstatus


1. In diesem Abschnitt sehen Sie den Status aller Tabellen. Wenn Sie auf eine grüne Tabelle klicken, wird diese rot. In diesem Fall haben Sie also eine Bestellung für den jeweiligen Tisch gebucht

05. Verwaltung der empfohlenen Produkte


1. Ändern Sie die Position des empfohlenen Produkts, indem Sie auf die Aufwärts- oder Abwärtspfeile klicken.

2. Fügen Sie ein neues empfohlenes Produkt hinzu, indem Sie unten rechts auf das Pluszeichen klicken.

3. Sie müssen das Produkt, das Foto, den Preis, den optionalen Preis und die Rangfolge auswählen.

4. Um ein empfohlenes Produkt zu ändern, müssen Sie auf die Schaltfläche Bearbeiten klicken.

5. Sie können das Foto, den Preis, den optionalen Preis sowie die Rangfolge ändern.

5. Sie können ein empfohlenes Produkt löschen, indem Sie auf die Schaltfläche Löschen klicken.

06. Tabelle Änderungsanforderung



07. Anzeige von Trinkgeld durch Kunden



08. Verwalten Sie kostenlose Produkte abhängig von der Anzahl der Bestellungen.



1. Sie können kostenlose Produkte aus der Produktliste des Restaurants hinzufügen.

2. Sie können die Produkte kostenlos aktivieren oder deaktivieren, indem Sie auf die Schaltfläche oben klicken

3. Sie können ein neues Produkt hinzufügen, das nicht in der Liste der Restaurantprodukte vorhanden ist, indem Sie auf die Schaltfläche "Fügen Sie ein kostenloses Produkt Produkt klicken" klicken.

4. Sie müssen den Produktnamen eingeben und dann speichern.

09. Altersbegrenzungsmanagement für Produkte


Sie können die Altersgrenze für Kunden für jedes Produkt festlegen, indem Sie auf die Nummer 16 oder 18 klicken

10. Verwaltung von Gutschein


Sie können einen vorhandenen Coupon aktivieren oder deaktivieren, einen neuen Coupon hinzufügen, einen vorhandenen ändern oder einen vorhandenen löschen.



11. Takeaway - Management


Legen Sie den Arbeitsplan für Takeaway fest, indem Sie auf die Schaltfläche oben klicken

Geben Sie die angegebenen Stunden in das zu öffnende Formular ein

Sie können die Positionen der empfohlenen Produkte ändern, die nur im Menü zum Takeaway angezeigt werden. Sie können empfohlene Produkte auch löschen, indem Sie auf die Schaltfläche X klicken

Sie können alle Produkte aus der Takeaway-Produktliste löschen

Sie können alle Produkte aus der Restaurant-Produktliste mit einem Klick zur Produktliste zum Takeaway hinzufügen

Sie können ein neues Produkt zum Mitnehmen hinzufügen, indem Sie auf die Schaltfläche "Fügen Sie ein neues Produkt zum Mitnehmen hinzu" klicken

Sie müssen den Produktnamen, die Beschreibung, den Preis und den optionalen Preis nach 20:00 Uhr eingeben und die Produktkategorie auswählen

Sie können Produkte für Takeawy ändern, indem Sie auf die entsprechende Kategorie und dann auf das Stiftsymbol klicken

Sie können ein Foto für das empfohlene Produkt anhängen, den Namen, die Beschreibung, den Preis und den Preis nach 20:00 Uhr ändern

12. Produktmanagement für die Lieferung


Sie können Ihre Arbeitszeiten für die Lieferung verwalten, indem Sie auf die Schaltfläche "Lieferzeit" klicken

Sie können die Arbeitszeiten für die Lieferung festlegen

Sie können PLZ für Lieferung verwalten, indem Sie sie hinzufügen, löschen, aktivieren oder deaktivieren

Sie müssen die entsprechenden Felder ausfüllen, um eine PLZ hinzuzufügen.

Sie können alle Produkte mit nur einem Klick aus der Lieferproduktliste löschen, indem Sie auf die Schaltfläche Alles entfernen klicken.

Sie können alle Produkte aus der Liste der Restaurantprodukte mit nur einem Klick zur Liste der Produkte für Delivery hinzufügen, indem Sie auf die Schaltfläche "Füge alle Funktionen" klicken.
